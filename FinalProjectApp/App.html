Overview

The next project is a word prediction Shiny app. The app is the final project of the Data Science Capstone Course in Coursera. The objective is an application that predicts words using a natural language. We will need to input a text and the application will looked it up in the training corpus data and the most common word in its list of following words will be suggested similar to the technology of Swiftkey implemented on the smart phones.

Instructions

The app has 2 section. The first one is the instruction and review criteria for the final project submission tab. The second one is the word prediction model visualization there you can input a word or a simple phrase and you will receive the possible next word to complete the phrase.

Instruction and review criteria

This section is going to show you the instruction and the review criteria about the final project submission. You can use it as a guide to evaluate my app.

The submission consist in three parts:

  1. Repository project files where you can find the code used for the application
  2. Slide deck presentation which describe the algorithm used to build the model
  3. The final application which is a word predictor application

Word Prediction APP

The prediction model uses a tidy data applied to text mining in R. To building the model algorithm we considered the next steps:

  1. The training data comes from three types sources Twitter, Blogs and News files
  2. Cleaning and removing the non-alphanumeric's, url's and repeated letters
  3. Data separation in N grams (the model consider 2, 3, 4, 5 and 6 ngrams dataframes)
  4. Finally, selecting the most probably word to predict the next word of a phrase.

The user will have to write a word or a phrase in the input panel and then press the submit button to receive the next word.

Additional note: due to the large dataset, we use a 5% of the original data to train the model so there is a possibility that the user will need to enter another phrase if the application does not find a word.
